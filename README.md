# Shamela Books in English Translation and Arabic Original

A simple, fast, and offline-capable web app for browsing translated Arabic Islamic books from the Shamela library. This repository contains **Shamela Arabic books translated into English**, organized by category. Each book JSON file includes both Arabic and English content, along with metadata such as author, edition, category, and `new_page_number` for precise page tracking.
